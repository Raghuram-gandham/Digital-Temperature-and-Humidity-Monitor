# Digital-Temperature-and-Humidity-Monitor
An Arduino-based temperature and humidity monitoring system with:
- LCD display output
- Celsius/Fahrenheit toggle button
- Real-time sensor readings
## Features
- Real-time temperature monitoring
- Humidity sensing
- LCD display interface
- Celsius to Fahrenheit conversion (Vise versa)
- Push-button interaction
## Components
- Arduino Uno
- DHT11 sensor
- 16x2 LCD display
- Push button
- Breadboard and jumper wires
